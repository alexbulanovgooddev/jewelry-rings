# Jewelry Rings

Современный веб-сайт коктейльного бара.

## Быстрый старт

```bash
# Установка зависимостей
npm install

# Запуск dev-сервера
npm run dev

# Сборка для продакшена
npm run build

# Предпросмотр сборки
npm run preview
```

## Команды

| Команда                      | Описание                             |
| ---------------------------- | ------------------------------------ |
| `npm run dev`                | Dev-сервер с HMR                     |
| `npm run build`              | Проверка типов + production-сборка   |
| `npm run preview`            | Локальный просмотр production-сборки |
| `npm run lint`               | Запуск всех линтеров                 |
| `npm run lint:eslint`        | Проверка JS/TS через ESLint          |
| `npm run lint:eslint:fix`    | Автоисправление ESLint               |
| `npm run lint:stylelint`     | Проверка CSS/SCSS через Stylelint    |
| `npm run lint:stylelint:fix` | Автоисправление Stylelint            |
| `npm run lint:prettier`      | Проверка форматирования              |
| `npm run format`             | Форматирование кода через Prettier   |
